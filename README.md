# 🐦 Pidgeon Jump

Pidgeon Jump é um jogo 2D desenvolvido em **Python com Pygame**, inspirado no clássico Flappy Bird.

No jogo, você controla um **pombo voando por uma cidade em pixel art**, desviando de prédios e tentando alcançar a maior pontuação possível.

Este projeto foi desenvolvido com o objetivo de **estudar programação, estrutura de projetos e desenvolvimento de jogos utilizando Python**.

---

# 🎮 Gameplay

Controle um pombo que precisa desviar de obstáculos enquanto voa pela cidade.

- pressione **SPACE** para bater as asas
- desvie dos prédios
- acumule pontos
- tente bater o **recorde**

---

# ✨ Funcionalidades

- sistema de pontuação
- best score salvo em arquivo
- animação do pombo
- obstáculos gerados proceduralmente
- colisão com obstáculos
- menu inicial
- tela de game over
- música de fundo
- efeitos sonoros
- background com movimento

---

# 🛠 Tecnologias utilizadas

- **Python**
